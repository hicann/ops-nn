# HardSwish
## 产品支持情况

| 产品                                                         | 是否支持 |
| :----------------------------------------------------------- | :------- |
| <term>Atlas A2 训练系列产品/Atlas 800I A2 推理产品/A200I A2 Box 异构组件</term> | √        |
## 功能说明

- 算子功能：对输入张量x逐元素计算HardSwish激活函数值，计算公式为：
$y = \frac{x * min(max(x + 3, 0), 6)}{6}$。

## 参数说明

| 参数名 | 输入/输出/属性 | 描述                                                         | 数据类型 | 数据格式 |
| :----- | :------------- | :----------------------------------------------------------- | :------- | :------- |
| self        | 输入           | x        | float、half、bfloat16    |   ND      |
| out         | 输出           | y        | float、half、bfloat16    |   ND     |
