## 功能说明

- 算子功能：对张量列表中的每一个元素执行先加一再以e为底的对数函数运算。
- 计算公式：
  
  $$
  x = [{x_0}, {x_1}, ... {x_{n-1}}]\\
  y = [{y_0}, {y_1}, ... {y_{n-1}}]\\
  $$

  $$
  {\rm y}_i = {\rm log}_e(x_i + 1) (i=0,1,...n-1)
  $$
