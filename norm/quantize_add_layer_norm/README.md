## 功能说明

- 算子功能：是一种结合了量化、加法和层归一化的操作。依次完成以下操作：
  - 加法：将两个输入张量进行加法操作。
  - 层归一化：对加法后的张量进行层归一化操作。
  - 量化：将输入张量从浮点数量化为低精度整数或浮点数。

- 计算公式：

  $$
  x = x1 + x2 + bias
  $$

  $$
  rstd = {{1}\over\sqrt {Var(x)+eps}}
  $$

  $$
  norm_out = (x-\bar{x}) * rstd * gamma + beta
  $$

  $$
  out = round((norm_out/scales)+zero_points)
  $$
